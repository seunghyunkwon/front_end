css는 html 문서의 스타일을 지정하는데 사용하는 언어입니다. css는 html 요소가 어떻게 표시되어야 하는지 설명합니다

CSS란?

  • CSS는 Cascading Style Sheets의 약자이다.
  • CSS는 한번에 여러 웹페이지(html파일)에 적용될 수 있다.
  • 확장자 css는 별도로 외부파일에 css를 작성할 때 사용한다.
  • CSS작성은 style 태그 사용한다. 특정 요소에 적용할 선택자(selector)가 필요하다.
  • CSS 프로퍼티는 글꼴 크기, 색상, 배경이미지, 테두리 등이 있으며 모든 태그에 공통적인 것 또는 특정 태그에만 적용되는 것을 구별해서 사용한다.
